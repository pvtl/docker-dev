# Meilisearch

We use the official [Meilisearch](https://hub.docker.com/r/getmeili/meilisearch/) Docker image.


## Connecting

| Parameter | Value |
|-------------|---|
| Host | `meilisearch` (from a container)<br>`localhost` (from your computer) |
| Port | `7700` |
| Password | `meilisearch_admin_key` |
